반응형 웹(Responsive Web)

반응형 웹 디자인이란 하나의 웹 사이트에서 PC, 스마트폰, 태블릿 PC 등 접속하는 디스플레이의 종류에 따라 화면의 크기가 자동으로 변하도록 만든 웹 페이지 접근 기법

  • 화면 설계 시 특정 뷰 포트 너비에 의존하지 않는다.
  • 큰 고정 너비 요소를 사용하면 가로 스크롤이 발생할 수 있으므로 유의한다.
  • 상대 너비를 사용하고 절대 위치의 값이 너무 크지 않게 조정한다.
  • CSS 미디어 쿼리를 사용한다.
