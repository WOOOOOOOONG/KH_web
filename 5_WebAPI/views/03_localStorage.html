localStorage

웹 스토리지가 나오기 전(HTML5 이전)에는 쿠키(cookie)를 사용했으나 저장 용량이 적고 보안에 취약했음
웹 스토리지는 사용자 측에서 좀 더 많은 양의 정보(최소 5MB 이상의 공간)를 안전하게 저장할 수 있도록 하며
이 정보는 절대 서버로 전송되지 않는다.

- localStorage : 보관 기한이 없는 데이터를 저장한다.
브라우저 탭이나 창이 닫히거나, 컴퓨터를 재부팅 해도 저장 된 데이터는 없어지지 않는다.
- sessionStorage : 하나의 세션만을 위한 데이터를 저장한다.
사용자가 브라우저 탭이나 창을 닫으면 이 객체에 저장 된 데이터는 사라진다.
