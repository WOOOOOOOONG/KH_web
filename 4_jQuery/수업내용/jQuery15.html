시각적 효과

Effect 메소드 : 페이지에 애니메이션 효과를 만들기 위한 메소드 집함

    1. $('선택자').메소드명();
    2. $('선택자').메소드명([speed]);
    3. $('선택자').메소드명([speed], [easing], [callback]);
    - speed : 실행 속도(밀리세컨즈) / 숫자 or slow, fast
    - easing : 변경되는 지점별 속도 / linear, swing 가능
    - callback : 메소드 실행 후 실행할 함수

show()와 hide()

문서 객체를 크게 하며 보여주거나 작게하며 사라지게 한다

$('선택자').show(speed, easing, callback); $('선택자').hide(speed, easing, callback);
